Instructions for Game Play

A linked list is a linear collection of data elements, whose order is not given by their physical placement but instead each element points to the next.
Ready to learn about Linked Lists? Here are the rules:
  • There will be an empty linked list to start off the game. When you enter, a random number will light up, this is the number you are going to insert in the list.
  • You will then move the cursor underneath the linked list to where you think this element will be inserted, thereby showing that you understand how the pointers work.
  • Finally we will check to make sure you put the cursor in the right place and if youa are right you will keep going and if you are wrong you will lose time!

Why Linked Lists?



Dynamic Size & Ease of Insertion/Deletion

Linked lists store linear data like arrays do. However arrays are limited because their size is fixed. One must know the upper limit on the number of elements in advance, and inserting a new element in an array of elements is expensive because room has to be created for the new elements, and to create room for existing elements that have shifted.

Applications



Our personal computers. All the running applications are kept in a linked list and the OS gives a fixed time slot to all for running. The operating system keeps on iterating over the linked list until all the applications are completed.
